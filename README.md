
Credit Risk & Loan Analysis Project

 About This Project

This project is all about understanding loan behavior and credit risk using real-world style financial data. I worked on cleaning the data, exploring patterns, and extracting meaningful insights that can help in making better lending decisions.

I used Python for data cleaning and EDA, and SQL for deep data analysis and business insights.

---
What I Tried to Achieve

The main idea behind this project was simple:

Understand which loans are risky
 Find patterns in customer repayment behavior
 Analyze how interest rate and loan type affect defaults
 Turn raw data into useful business insights


---


 How the Project Works
 
 
 Data Cleaning (Python)



 Removed duplicates and missing values
 Fixed incorrect data types
 Standardized inconsistent values
 Converted date formats for analysis

---

 Exploratory Data Analysis (EDA)

Then I explored the data visually and statistically:
 Loan distribution patterns
 Default vs non-default behavior
 Interest rate impact on loan status
 Customer segmentation based on loan behavior
 Risk trends across different loan types

---
SQL Analysis

After Python analysis, I used SQL to dig deeper:

 Total loan amount per customer
 Loan performance by type
 Closed vs active loans
 Revenue insights from interest rates
 Identification of high-risk customers

---
 Key Insights I Found

Some loan types are significantly more risky than others
 Higher interest rates often show stronger correlation with defaults
 A small group of customers contributes to most of the risk
 SQL aggregation helped reveal business-level insights clearly
